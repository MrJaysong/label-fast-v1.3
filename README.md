# 介绍
Labelfast 标注工具近期进行了功能更新，在菜单栏中新增了“数据集（Dataset）”选项。此次更新主要增加了四个实用功能，旨在帮助用户更高效地完成标注后的数据整理、划分和预处理工作。以下是这四个新功能的具体介绍。
